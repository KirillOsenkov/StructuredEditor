Formatting of the HTML documentation

Tags

The following tags are allowed:

  • <p> <span> <div> <pre>
  • <h1> <h2> <h3>
  • <ul> <ol> <li>
  • <table> <tr> <td>
  • <br/> <hr/>
  • <img>
  • <a>
  • <b> <i> <blockquote>

Styles

Please use the stylesheet docs.css.

Use class "note" for remarks.

Use tag "pre" and class "code" for 
multiline code fragments,
    with preserved tabulation.

Use tag "span" and class "codeword" to mark single words as code.

All links should be relative, i.e. no absolute links.
